# Dialect Web Project

## Overview
The Dialect Web project is designed to provide an interactive and informative guide to Chinese dialects. It presents structured information from a dialect memory guide in a user-friendly web format.

## Project Structure
```
dialect-web
├── public
│   └── index.html        # Main HTML document
├── src
│   ├── styles
│   │   └── style.css     # Styles for the webpage
│   └── scripts
│       └── app.js        # JavaScript for dynamic behavior
├── package.json           # npm configuration file
└── README.md              # Project documentation
```

## Installation
1. Clone the repository:
   ```
   git clone <repository-url>
   ```
2. Navigate to the project directory:
   ```
   cd dialect-web
   ```
3. Install the dependencies:
   ```
   npm install
   ```

## Usage
To start the project, open the `public/index.html` file in your web browser. This file serves as the entry point for the application and displays the content derived from the dialect memory guide.

## Features
- Structured tables for dialect information.
- Spatial memory maps for better retention.
- Memory techniques to aid learning.
